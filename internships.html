Internships

Previous and future internship expierences.

Summer 2018: Microsoft -- Completed Internship

For the duration of summer 2018 I was an explore intern at Microsoft. As an explore intern we are able to try the software engineer role as well as the program management role. The program is built for development so we are paired with two other interns for the project. I worked under Azure, Microsoft's cloud sector within Azure PIE (Product Infrastructure Engineering).

Our project for the summer was to create an in-app expierence for Microsoft's Incident Management platform. The platform is currently only internal, but in preperation of possibly making it a public we were assigned to make an in- app expierence to improve usability. In-app expierence was defined by our team as an expierence in which a user who is not familiar with the platform at all would be able to follow guided interactive tours developed by our team to self teach themselves to become proficient in the program. As the user became more proficient the tours available would increase to more advanced topics. We completed this goal within our time restraint setting up a template the team could easily duplicate and change to easily create in-app tours as well as creating the basic tours for new users.
